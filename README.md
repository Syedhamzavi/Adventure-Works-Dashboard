📊 Adventure Works Sales & Performance Dashboard | Tableau

An interactive Tableau dashboard built using the Adventure Works dataset to analyze sales performance, customer behavior, product performance, and regional trends. The dashboard provides actionable business insights through interactive visualizations and key performance indicators.

🚀 Project Overview

This dashboard helps users monitor business performance by analyzing:
- Sales
- Profit
- Orders
- Customers
- Products
- Regions
- Monthly, Quarterly, and Yearly Trends

📊 Dashboard Features

- KPI Cards (Orders, Customers, Sales, Profit)
- Top 5 Products
- Top 5 Customers
- Top 5 Regions
- Sales vs Production Cost Combo Chart
- Year-wise Sales Analysis
- Quarter-wise Sales Distribution
- Month-wise Sales Trend
- Interactive Filters

🛠️ Tools & Technologies

- Tableau Desktop
- Adventure Works Dataset
- Calculated Fields
- Interactive Dashboards

📈 Key Insights

- Total Sales: ₹29.36M
- Overall Profit: ₹12.08M
- Total Orders: 60.40K
- Total Customers: 18.48K
- Australia recorded the highest sales.
- Sales peaked in 2013.
- Q4 generated the highest quarterly sales.
